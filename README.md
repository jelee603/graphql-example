# graphql-example

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
http://localhost:8080
```

### Run graphQL Server
```
cd server
node index.js
http://localhost:4000/graphql
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
